# CancerDAO Cross-Border Medical Assistant

跨境医疗助手 skill。整合香港和内地的病历为完整时间线，提供药物可及性对比和费用估算。

## Install

```bash
npx skills add CancerDAO/cross-border-medical-assistant
```

## When to use

- 在香港和内地都看过病，需要整合
- 比较香港和内地的治疗方案、费用
- 查询药物在香港/内地的上市状态和价格

## Key features

- 统一病历时间线（JSON格式）
- 药物可及性对比（NMPA vs HK DOH）
- 费用估算（参考范围）
- 治疗方案比较
